day7

字符串

  • ASCII :48 ------ ‘0’
  • ASCII :65 ------ ‘A’
  • ASCII :97 ------ ‘a’

fgets && getline

fgets(s,100,stdin) s为字符数组，100为最多读入量，stdin固定

getline(cin,s) 此时s为String

用法不一样

  • 如果想读入到字符数组还有一个办法
  • cin.getline(s,100)

puts

输出字符数组

puts(s);

一些函数

  • 引入头文件#include < string.h> 或 #include < cstring>`

strlen(s);

求字符串的长度

strlen（s）

strcmp(a,b)

比较字符串大小，a<b，返回-1；a == b，返回0；a>b，返回1

strcpy(a,b)

将字符串b复制给a

String

  • 在iostream库里

默认：string s1;

复制：string s2 = s1;

赋值：string s3 = "hello"

重复：string s4(10,c) cccccccccc

  • 读写字符串

string s1,s2;

cin >> s1 >> s2;

cout << s1 << " "<<s2 << endl;

string写入只能用cin不能用scanf，输出可以用printf(%s ,s1.c_str()),也可以用puts(%s ,s1.c_str())
  • 如果要读入一整行

getline(cin,s1);

  • 重要函数

s1.empty()

判断函数是否为空 空则输出1

s1.szie()

字符串长度，速度比strlen快

string 变量可以直接比较运算，相加为拼接在一起

  • 遍历

for(char c : s) cout << c << endl; C++的特殊写法

字符串加密
