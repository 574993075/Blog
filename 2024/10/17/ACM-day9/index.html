字符串输出

题目

  • 输出字符串

我的方法

  • 字符和它紧邻后的字符ASCII码值相加，尾和头相加，一个环，此时用int(s[i])+int(s[(i+1)%s.size()])
  • 将每个字符转换成int类型相加，之后转换成char型再加到string中

单词替换

题目

我的方法(错误)

思路

  • 接收改变值和被改值
  • 遍历字符串中每个字符
  • 如果有被改值.size()个相同并且后面一位是空格则找到被改值
  • 对被改值前和后分别存放在pro和back
  • 将pro，改变值，back连接在一起
  • 输出

问题

单词匹配逻辑不正确
只是比较了字符，而不是完整的单词

数组越界错误
在内层循环中 if (count == bc.size() && s[i+1] == ' ')， s[i+1] 这个位置可能会导致数组越界

字符串拼接有问题
在构建新的字符串 pro 和 back 时，没有正确初始化这些字符串。应该使用 string::substr 来获取子字符串，而不是通过 for 循环构建字符串

改进

  1. 使用 s.find(bc, pos) 来找到 bc 单词的位置。
  2. if 条件检查前后是否是空格，确保是完整单词匹配。
  3. 使用 s.replace 来替换单词，并且更新 pos 指针位置，避免重复替换。
  4. string::npos 是 C++ 标准库中的一个特殊值，用于表示 查找操作失败 的情况,一般为-1

sstream方法

  • 头文件<sstream>

  • 用法：stringstream ssin(s)这里s为string类型

stringstream是把字符串变成字符串流，ssin相当于cin，只不过cin从键盘输入流，ssin从s这个字符串里输入流，使用方法直接用ssin>>赋值就好了，遍历字符串中的值，遇到空格或回车就输出流给赋值
  • 头文件<algorithm>
  • swap(a,b)此时会互换string类型的a、b的值
